A suitable Python envirnoment can be installed via anaconda environment manager with the aid of the requirements or spec-file.

The Runs folder withholds trained models and statistics
The Program files folder contains all that is needed to run or re-run the software

It is adviced to load the program files into the IDE of choice, e.g. Pycharm to 
   view the "Cloudpoint determination.ipynb" jupyternotebook for dataset insights 
 or
   use the neural network via cloudpointprediction.py pythonfile (modes to use or train are set at the end of the file).

The default export path is the users documents path + "jupyter files".

There are two versions of the Dataset. Use the curated dataset in the program files folder when running the software
